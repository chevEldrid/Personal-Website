What is React?

First: some setup. Since React is a JavaScript library it's not exactly a software we can simply download and make use of. Instead, we need to string together smaller bits of software to create a React environment we can then access from our favorite IDE or text editor. For setup, I'll turn it over to the folks at CodeCademy who do a fantastic job explaining how to install React on your own machine, and you can find this tutorial here. React is a JavaScript library focused on making User Interfaces with as little hastle as possible. These interfaces can then be used to create anything from a website to a mobile app. This also means one set of code works on computers, Androids, and iPhones. React doesn't do favorites. The library also supports a JavaScript Syntax called JSX, an updated take on vanilla JavaScript that allows for a more HTML look and feel to the written code. This can help you visualize what the JavaScript will end up implementing into your html files since it's spelled right out in the JSX.

JSX can return something resembling HTML to then populate an html page! This means provided we format the JSX correctly, we could send almost anything to be directly pulled into our webpage! We just have to remember to include the "document.querySelector('.container'));" at the bottom telling our JSX code to look for a container class and give it the result of "App".
