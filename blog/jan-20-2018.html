AWS and Swanky Features

January 20, 2018

It's been a crazy few weeks with constant AWS training and some light traveling to Boston and back, but in little over a week it should all be worth it! My Systems Architect Associate exam is scheduled for January 30th. I've finished the course so now it just comes down to some diligent studying...

AWS

I've spent more time staring at this screen in the past two weeks than I have watched TV in the past two months. Parts of me cringe at that fact but only one of those two activites allows me to set up VM intances that can handle most any catastrophe or build out autoscaling web applications. AWS - 1 : Netflix - 0.

Probably the coolest process I've learned in the AWS platform recently is Route53: Amazon's DNS service. Since Amazon is now a domain registrar not only can I build out websites either within the S3 file system or in an EC2 virtual server, I can also buy domain names and store all their relevent certificates in the same interface! So, I went ahead and bought cheveldrid.com, hoping to someday soon move this site over to that more classic-style domain name. Or maybe use it for something completely different! So far it's been fantastic practice to build quick websites using an actual domain I can access from various devices. This enables me to see if all the work I'm doing behind the scenes really does look like it should as a final project with relative ease.

Website Updates

So, like last time introduced the change to blog entries...this week we're seeing some bigger adjustments to this website as a whole. Looking up, you'll see the addition of a new tab called blog where you must have gone to see this post. In an effort to better display the recent projects I've worked on and continue this blog series, I decided to put the projects first, with these status updates available deeper within the site. We'll see if these changes are for the better but I do really enjoy the cleaner look of the new landing page.

This shouldn't be the last of work I have planned for the immediate future, but is probably the biggest visual change. The rest will happen more behind the scenes to clean up the ever-growing code base, furthering language study, and increase automation.

That's about it! Relatively boring couple of weeks with so much time devoted to studying but coming soon: Snowfall updates!
