Weekly Update

January 5, 2018

I did not expect so much difficulty trying to pick this back up again, even after only a couple weeks but...so many undocumented idiosyncrasies that I had simply forgotten about. Whether it be metadata in the header that needs adjusting or making sure to add the correct post to the "prev" and "next" links at the bottom of each blog page. There's definitely an argument for using something like Wordpress. I have begun to look towards Jekyll as a way to automate some of this more finicky stuff. I love doing everything by hand but you can really only justify being stubborn for so long before accepting help from the tools that already exist to do exactly what you're looking for.

The New Format

So, if you hadn't noticed there's a bit of a different look to this new entry. Primarily, the title is now a generic "Weekly Update" as opposed to something somewhat related to the innards of the post. While it does mean I won't have to be quite so creative all the time, the real purpose is to standardize the output of entrees into a weekly sort of update bulletin. While before, I would tend to wait until I thought I had something worthy of an entire article, what I'm going to try and do now is have little paragraphs about each of the week's developments in a singular post published every Friday. Not only will this make it easier to blast myself with reminder updates, but it will also help with consistency. Okay, enough meta.

AWS

At the urging of a close friend, I bought a course on AWS, a cloud technology that is becoming more and more important to developers and IT professionals every day. I am currently learning the material needed to pass the "Solutions Architect" associate exam, which has a primary focus on using the cloud services provided by Amazon for system administration and infrastructure. While this isn't exactly what I want to end up doing, it's been incredible at connecting some of the Data management work I had done at GMO this past semester and the more computer science side of systems management.

Using the free tier of AWS, which gives you limited access to some of the more advanced features and potential higher fees you still have a completely endless cloud playground at your fingertips. Amazon does a great job providing tutorial videos and posts about how to do the many things a developer or anyone interested in computer science might find useful. So far, I've worked primarily with S3: Amazon's standard cloud storage platform. If you look closly at the screenshot too you might see the link to a tutorial on building a static site. That is my current goal for the platform: learn what I need to get certified but also move this website to a more standard hosting service that has more relevance in the workforce.

Chromebook

For the past few years, my computer of choice was a pretty beefy MSI gaming laptop. It's a fantastic machine, and has done amazing with staying relevant...but...it's a monster with not so great battery. To this end I started to look for a uhh more portable, portable computer and ended up between a Macbook Pro and a Chromebook. Besides the obvious price factor, it really came down to ability to program in a linux-esque environment and not be tied to an outlet. So I did my research, polling friends about their machines of choice and scouring the net for things like this and this. My overall conclusion was that while the Chromebook might require a little more setting up, installing terminal applications and understanding the relationship between android apps and chrome extensions, it could be a competitive choice for relatively little risk. Chromebooks' default configuration includes directories for documents and downloads but the file manager works more as a UI for your linked Google Drive account. So I went for it, this latest post has actually been the first real test of the little Chromebook that could!

Next week I hope to give it a little more of a programming test and along with regular web dev. This isn't meant to replace my other laptop, not yet anyway, just offer a portable solution for the more easy to compute and create assignments as well as for general use. So far, I've been loving it.
